# CleanArchitectureMVVM-2021
드로이드나이츠를 보고 기능별로 모듈을 작은 단위로 나누어 작업해보는 새로운 스타일을 스터디하고자 만들었습니다.

## 어떤것을 할것인가?
- Kotlin DSL
- 기능별로 모듈 작게 나누기
- Compose 적용
- Coroutine 적용
